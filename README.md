# Laoues

## A General-Purpose Proxy Tool

> Written in Go, configuration files use TOML format
>
> License: GPL-V3 by default unless otherwise specified

[Official Website](https://867678.xyz/projects/moixa) [简体中文版](moixa介绍.md)

## Quick Start

> Similar configuration format to Clash. If you are familiar with Clash, migrating to Moixa will not be a problem.
>
> Provides binary files and sample configuration files. For details on adding nodes, please refer to the official website.
>
> Currently supports all platforms supported by Go.
>
> Download the binary files from Tags.
